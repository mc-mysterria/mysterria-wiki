Towny - це плагін, що дозволяє гравцям створювати та управляти містами, захищати свою територію та взаємодіяти з іншими поселеннями.

## Основи Towny

### Створення міста
```
/town new <назва_міста>
```

### Основні команди міста
- `/town` - інформація про ваше місто
- `/town spawn` - телепорт до спавну міста
- `/town invite <гравець>` - запросити гравця до міста
- `/town kick <гравець>` - виключити гравця з міста

### Управління ділянками
- `/plot claim` - захопити ділянку
- `/plot unclaim` - відмовитися від ділянки
- `/plot set <тип>` - встановити тип ділянки
- `/plot toggle` - змінити налаштування ділянки

## Типи ділянок

### Житлові ділянки
- Для будівництва будинків
- Можна продавати іншим гравцям

### Комерційні ділянки
- Для магазинів та бізнесу
- Вища ціна за ділянку

### Спеціальні ділянки
- Ферми, склади, громадські споруди

## Економіка міста

### Податки
- Встановлюйте розумні податки
- Баланс між доходом та привабливістю

### Бюджет
- Витрати на утримання міста
- Розширення території

## Дипломатія

### Союзи
- `/town ally add <місто>` - запропонувати союз
- Спільна оборона та торгівля

### Вороги
- `/town enemy add <місто>` - оголосити ворогом
- PvP на нейтральній території

Детальнішу інформацію дивіться в розділі економіки та політики!
