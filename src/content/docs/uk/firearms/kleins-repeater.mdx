## Огляд

Повторювач Кляйна - це збалансована, надійна вогнепальна зброя, ідеальна для новачків і достатньо універсальна для досвідчених користувачів. Це **єдина зброя, яку ви можете створити з нуля**, що робить її вашою браною до містичної вогнепальної зброї.

---

## Характеристики

| Атрибут | Значення |
|-----------|-------|
| **Тип** | Містичний повторювач |
| **Урон** | 4.0 за постріл |
| **Розмір магазину** | 6 патронів |
| **Дальність** | 40 блоків |
| **Час перезарядки** | 3 секунди (повний магазин) |
| **Швидкість стрільби** | 1 постріл за секунду |
| **Тип боєприпасів** | Патрони Кляйна (підтримуються всі 13 варіантів) |
| **Тип перезарядки** | Повний магазин |
| **Точність (стоячи)** | Ідеальна (0° розкид) |
| **Точність (в русі)** | 4° розкид |
| **Точність (в повітрі)** | 8° розкид |

## Переваги та недоліки

### ✅ Переваги

- **Створюється з нуля** - Не потрібно фармити підземелля
- **Збалансовані характеристики** - Підходить для більшості бойових ситуацій
- **Швидка стрільба** - 1 постріл за секунду дозволяє робити швидкі повторні постріли
- **Помірна дальність** - 40 блоків покривають більшість бойових зіткнень
- **Легко обслуговувати** - Проста крафтова боєприпасів

### ❌ Недоліки

- **Нижчий урон** - Лише 4.0 за постріл (найнижчий серед усієї зброї)
- **Середня дальність** - Поступається гвинтівці на великих відстанях
- **Перезарядка повного магазину** - Неможливо перервати перезарядку для стрільби
- **Менший магазин** - 6 патронів проти тактичної перезарядки Деткнелла

## Гід по бою

### Найкраще застосування

**Бій на близькій та середній дальності (10-40 блоків):**
- Швидка стрільба для тривалого урону
- Ефективний проти груп з вогняними патронами
- Підходить для мобільного бою

**PvP сценарії:**
- Патрони контролю духу для знешкодження
- Патрони обману для дезорієнтації
- Імплозивні патрони для сплеску урону

**Прогресія на початку гри:**
- Перша зброя до отримання винагород з підземель
- Надійний вихідний урон для дослідження
- Легке підтримання боєприпасів

### Тактичні поради

<Aside type="tip">
**Управління перезарядкою:** Повторювач Кляйна має 3-секундну повну перезарядку. Знайдіть укриття або створіть дистанцію перед перезарядкою!
</Aside>

**Перед боєм:**
- Зарядіть імплозивні або екзорсистські патрони для перших пострілів
- Переконайтеся у повному магазині перед боєм
- Розташуйтеся на 20-35 блоках для оптимальної точності

**Під час бою:**
- Стійте нерухомо при стрільбі для ідеальної точності
- Використайте всі 6 патронів перед перезарядкою
- Перемкніться на ближній бій якщо попалися під час перезарядки

**Вибір боєприпасів:**
- **Загальне дослідження:** Стандартні патрони Кляйна
- **Полювання на нежить:** Екзорсистські патрони (250% урону)
- **Броньовані вороги:** Бронебійні патрони
- **Боси:** Імплозивні патрони (125% урону)

## Рекомендовані комплекти спорядження

### Комплект дослідника
- **Основні боєприпаси:** 2 стеки патронів Кляйна
- **Спеціалізовані:** 1 стек екзорсистських патронів (нежить)
- **Спеціалізовані:** 1 стек вогняних патронів (площинний урон)

### PvP комплект
- **Відкриття:** 12 імплозивних патронів (сплеск урону)
- **Контроль:** 1 стек патронів контролю духу
- **Дебаф:** 1 стек патронів позбавлення
- **Запасні:** 1 стек стандартних патронів Кляйна

### Комплект мисливця на босів
- **Основні:** 2 стеки імплозивних патронів
- **Дебаф:** 1 стек патронів послаблення
- **Броньовані:** 1 стек бронебійних патронів

## Часті питання

**П: Чи можу я використовувати патрони Деткнелла у Повторювачі Кляйна?**

В: Ні, Повторювач Кляйна приймає лише патрони Кляйна. Однак усі 13 спеціальних ефектів боєприпасів працюють з патронами Кляйна!

**П: Скільки урону завдає Повторювач Кляйна з імплозивними патронами?**

В: Імплозивні патрони завдають 125% урону, тобто: 4.0 × 1.25 = **5.0 урону за постріл**

**П: Чи підходить Повторювач Кляйна для PvP?**

В: Так! Хоча він має нижчий урон, швидка стрільба та підтримка спеціальних боєприпасів (контроль духу, обман, позбавлення) роблять його ефективним у PvP.

**П: Чи можу я перервати перезарядку?**

В: Ні, Повторювач Кляйна використовує механіку перезарядки повного магазину. Ви не можете стріляти, поки не завершиться 3-секундна перезарядка.

**П: Який найкращий тип боєприпасів для Повторювача Кляйна?**

В: Це залежить від вашої ситуації:
- **Екзорсистські патрони** для нежиті (10.0 урону проти нежиті!)
- **Імплозивні патрони** для босів (5.0 урону)
- **Патрони контролю духу** для PvP
- **Стандартні патрони Кляйна** для загального використання
