### Що таке Містичне Озброєння?

Містичне Озброєння переносить легендарну вогнепальну зброю з «Lord of the Mysteries» у Minecraft! Створюйте потужну зброю, заряджайте її спеціалізованими боєприпасами та відчуйте абсолютно новий вимір бою.

### Що вам знадобиться

Щоб почати роботу з містичною вогнепальною зброєю:

1. **Верстак Зброяра** — спеціальна станція для складання зброї.
2. **Деталі зброї** — компоненти для створення вашої зброї.
3. **Боєприпаси** — кулі для стрільби.

### Як використовувати зброю

**Основні елементи керування:**
- **Права кнопка миші (ПКМ):** Стрільба.
- **Ліва кнопка миші (ЛКМ):** Перезарядка.

**Стрільба зі зброї:**
1. Тримайте зброю в руці.
2. Переконайтеся, що боєприпаси завантажені.
3. Прицільтеся в ціль.
4. Натисніть ПКМ, щоб вистрілити.
5. Слідкуйте за панеллю дій для відображення кількості патронів.

**Розуміння інтерфейсу:**

Панель дій (над вашим хотбаром) показує:
- **Поточні патрони / Макс. патрони** (наприклад, "Klein's Repeater 4/6").
- **Статус перезарядки** (жовте "Reloading..." під час перезарядки).
- **Тип патронів** (кольоровий текст, що показує ефект поточних боєприпасів).
- **Спеціальні попередження** (червоне "No ammo!", коли патрони закінчилися).

### Бойові механіки

**Система точності:**

Ваша точність змінюється залежно від вашого руху:

- **Стоячи на місці:** Ідеальна точність (розліт 0°).
- **Під час руху:** Розліт куль 4°.
- **У повітрі (стрибок/падіння):** Розліт куль 8°.

<Aside type="tip">**Точність має значення:** Стійте нерухомо для найкращої точності під час важливих пострілів!</Aside>

**Механіки перезарядки:**

Різна зброя має різні системи перезарядки:

**Повна перезарядка магазину (Репітер Клейна):**
- Заряджає всі патрони одночасно.
- Час перезарядки — 3 секунди.
- Неможливо перервати — потрібно дочекатися повної перезарядки.

**Послідовна перезарядка (Револьвер Смертного Дзвону та Парова Гвинтівка):**
- Заряджає по одному патрону за раз.
- Можна перервати перезарядку, щоб вистрілити.
- Тактична перевага — можливість часткової перезарядки.

### Поради для бою

**Керування боєприпасами:**
- Носіть із собою кілька типів патронів для різних ситуацій.
- Використовуйте патрони Екзорциста для Незеру та нежиті.
- Зберігайте Імплозивні патрони для босів та сильних ворогів.

**Час для перезарядки:**
- Репітер Клейна: знайдіть укриття і перезаряджайтеся в безпечні моменти.
- Револьвер/Гвинтівка: можна перервати перезарядку, щоб вистрілити у разі загрози.

**Візуальний зворотний зв'язок:**
- Кольорові сліди частинок показують траєкторію польоту кулі.
- Ефекти влучання в істот та блоки.
- Звуки прольоту куль при промахах поруч із вами.

**Керування дистанцією:**
- Використовуйте Репітер Клейна для ближнього та середнього бою.
- Револьвер Смертного Дзвону найкраще підходить для середніх дистанцій.
- Парова Гвинтівка домінує на далеких дистанціях.
