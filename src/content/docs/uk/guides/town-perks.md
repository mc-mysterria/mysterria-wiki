### Що таке Міські Переваги?

Міські Переваги — це престижна система, створена для нагородження гравців, які створюють виняткові архітектурні роботи та вражаючі споруди. Завдяки будівництву значних об'єктів, гравці можуть отримати спеціальні бафи та бонуси, які застосовуються до визначених територій всередині або поблизу їхніх міст. Ці покращення можуть поліпшити різні аспекти геймплею, включаючи сільське господарство, бойову ефективність, крафтинг та унікальні здібності.

### Як запросити Міську Перевагу

Щоб розглянути вашу заявку на Міську Перевагу, будь ласка, виконайте наступні кроки:

1. Побудуйте значну та вражаючу будівлю або споруду в вашому місті або поблизу нього
2. Переконайтеся, що ваша будівля має чітку, цілісну тему, яка доповнює загальну естетику вашого міста
3. Подайте заявку в підтримку з проханням розглянути вашу споруду для Міської Переваги
4. Адміністрація сервера оцінить вашу будівлю на основі:
   - Тематичної послідовності та творчого бачення
   - Масштабу, деталізації та якості будівництва
   - Інтеграції з навколишнім міським середовищем

Тип наданого бафу, разом з його параметрами та ефективністю, буде визначено командою адміністрації, щоб належним чином відобразити призначення вашої будівлі, тему та загальну якість.

### Доступні Типи Бафів

Наступні типи бафів можуть бути надані на основі характеру та теми вашої споруди:

#### Захист від Мобів
Запобігає появі ворожих мобів на визначеній території. Цей баф особливо підходить для мирних садів, храмів, зон для медитації або житлових безпечних зон, де спокій є найважливішим.

#### Ефекти Зілля
Надає безперервні корисні ефекти зілля всім гравцям, присутнім на території. Ефекти можуть включати підвищену швидкість, збільшену силу, регенерацію або будь-який інший відповідний ефект зілля. Цей баф ідеально підходить для тренувальних майданчиків, містичних святилищ або зачарованих локацій.

#### Покращене Сільське Господарство
Значно прискорює швидкість росту всіх культур, посаджених на території. Цей баф найбільш доречний для ферм, сільськогосподарських районів, ботанічних садів або друїдських гаїв, присвячених землеробству.

#### Відновлення Броні
Поступово відновлює міцність екіпірованої броні, поки гравці перебувають на території. Цей баф особливо підходить для ковальських районів, арсеналів, укріплень або священних місць під божественним захистом.

#### Підвищення Досвіду
Множить усі очки досвіду, отримані з будь-якого джерела, поки гравець перебуває на території. Це покращення ідеально підходить для бібліотек, академій, тренувальних закладів або пам'ятників знанням та навчанню.

#### Покращення Акторства
Збільшує швидкість акторства для Бейондерів, присутніх на території. Цей спеціалізований баф найбільш підходить для містичних локацій, кімнат для медитації, духовних місць або місць надприродного значення.

#### Духовна Регенерація
Безперервно надає очки акторства Бейондерам з плином часу, поки вони перебувають на території. Цей баф найкраще застосовувати до храмів, святилищ, священних земель або інших місць глибокого духовного значення.

### Покриття Території

Коли баф надається вашій будівлі, адміністрація визначить зону дії, використовуючи одну з наступних конфігурацій території:

#### Кругова Зона
Круглий регіон, зосереджений навколо конкретної точки, що простягається назовні до визначеного радіусу. Ця конфігурація проста та ефективна для окремих будівель, монументів або централізованих споруд.

#### Прямокутна Зона
Прямокутний регіон, визначений двома протилежними кутовими точками, що створює територію у формі коробки. Цей метод добре підходить для міських районів, будівельних комплексів або структурованих забудов з визначеними межами.

#### Користувацький Полігон
Гнучка форма території, визначена декількома точками, що дозволяє створювати нерегулярні або органічні межі. Ця конфігурація ідеальна для областей з природними контурами, нерівномірними міськими планами або будівель, які слідують особливостям рельєфу.

#### Міська Територія
Баф застосовується до всіх існуючих меж міської території вашого міста. Цей підхід зручний для міських переваг і усуває необхідність ручного вибору території.

### Рекомендації для Розгляду

Плануючи свою будівлю та подаючи заявку, будь ласка, майте на увазі наступне:

- **Масштаб та Деталізація**: Більші споруди з більшою увагою до деталей мають більше шансів отримати потужніші конфігурації бафів
- **Тематична Відповідність**: Тип бафу повинен відповідати призначенню вашої будівлі. Наприклад, сільськогосподарський комплекс природно отримає Покращене Сільське Господарство, а не Захист від Мобів
- **Чітке Призначення**: Споруди з чітко визначеними функціями та ролями легше оцінити адміністраторам та підібрати відповідні бафи
- **Декілька Переваг**: Гравці можуть отримати декілька бафів для різних будівель, дозволяючи містам накопичувати різноманітні переваги з часом
- **Потрібне Завершення**: Будь ласка, переконайтеся, що ваша споруда повністю завершена перед поданням заявки на оцінку

Система Міських Переваг створена для визнання та нагородження відданості виключному міськобудуванню, заохочення творчості та розвитку яскравих, значущих гравецьких спільнот через відчутні ігрові переваги.
