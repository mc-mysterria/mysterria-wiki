Welcome to the Mysterria Minecraft server! This guide will help you get started quickly.

## First steps:

### 1. Connecting to the server
- **Server IP**: `mysterria.net`
- **Minecraft Version**: 1.20.1
- **Server Type**: Survival with modifications

### 2. Registration
- Execute the command `/register <password> <password>`
- Remember your password!

### 3. Basic commands
- `/spawn` - teleport to spawn
- `/help` - command list
- `/rules` - server rules
- `/kit starter` - starter kit

### 4. First tasks
1. Explore the spawn and read the information signs
2. Find a safe place to build
3. Create or join a town
4. Choose your magical Pathway

## Useful tips

- Always read the chat - there's lots of useful information there
- Don't hesitate to ask questions in chat
- Join the Discord community
- Learn the magic system gradually

## Need help?

If you have questions, contact players or administration in chat or Discord!
